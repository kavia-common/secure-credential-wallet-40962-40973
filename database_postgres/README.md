# Database (Dev: SQLite, Prod: Postgres)

This container uses SQLite for development while maintaining PostgreSQL schema parity.

Dev setup:
1. Run: ./init_db.py
2. Validate: ./test_db.py
3. Viewer: source db_visualizer/sqlite.env && cd db_visualizer && npm start

Env:
- .env is generated by init_db.py with:
  - DB_URL=sqlite:///.../myapp.db
  - SQLITE_DB=.../myapp.db

Migrations (Postgres):
- See migrations/001_init.sql
- Apply with: psql "$DB_URL" -f migrations/001_init.sql
